# 📁 Images Folder

This folder contains all the images and GIFs used in the GitHub profile README.

Upload the following images to this folder:

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `header.gif` | Header banner animation | 1000x200px |
| `footer.gif` | Footer banner animation | 1000x150px |
| `fire.gif` | Fire animation icon | 30x30px |
| `wave.gif` | Waving hand animation | 30x30px |
| `rocket.gif` | Rocket animation icon | 30x30px |

| `snake.gif` | Snake animation icon | 30x30px |
| `trophy.gif` | Trophy animation icon | 30x30px |
| `banner.png` | Profile banner | 1000x300px |
| `profile.png` | Profile picture | 200x200px |

## 🔗 How to Use in README

Use this URL format to reference images:

```
https://raw.githubusercontent.com/brutal-45/brutal-45/main/images/FILENAME.gif
```

Example:
```markdown
<img src="https://raw.githubusercontent.com/brutal-45/brutal-45/main/images/rocket.gif" width="30" alt="Rocket">
```
